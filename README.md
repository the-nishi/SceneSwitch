### 🔄 Scene Switching in JavaFX

A JavaFX-based application that demonstrates how to switch between multiple scenes in a desktop GUI. This project focuses on navigation, UI control, and event handling using FXML and controllers.

#### 🚀 Features

* Switch between multiple scenes dynamically
* JavaFX GUI built with FXML
* Controller-based event handling
* Clean separation between UI and logic
* Lightweight and easy-to-understand implementation

#### 🛠️ Tech Stack

* **Language:** Java
* **Framework:** JavaFX
* **UI Design:** FXML

#### 📂 Project Structure

* `controllers/` – Handles scene switching logic
* `views/ (FXML files)` – Defines UI layouts
* `Main.java` – Application entry point

#### 🎯 Purpose

This project was developed to:

* Learn scene navigation in JavaFX
* Understand event-driven programming
* Practice structuring GUI applications
